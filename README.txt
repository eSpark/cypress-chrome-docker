Cypress Chrome Docker
---------------------


Docker image based on https://github.com/cypress-io/cypress-docker-images/blob/master/browsers/chrome69/Dockerfile

It's Debian based with node v8 and latest stable Google Chrome, with some other additions.

This one installs cypress 3.1.5 globally

Usage
=====

Pull it down like this:

     docker pull espark/cypress-chrome-docker:cypress3.1.5
